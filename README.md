# python-cli-tinydb-simple

## Description
A demo of creating, inserting, and searching database.

## Tech stack
- tinydb

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
